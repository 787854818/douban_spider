# Douban Spider
豆瓣爬虫，爬取的内容包括：
 1. 用户的信息
 2. 事件的信息
# 数据格式
用户：
 -
| 属性 | 描述 | 示例 |
|--|--|--|
| id | 用户id |92497641|
| city | 城市 |广东广州|
| description | 描述 |人生至高的追求:做个有趣无害的人。|
| name | 用户名 |特立独行的金鱼|
| signature | 个性签名 |读书不是爱好，而是生活必需品|
| verified | 是否认证 |false|
| verified_type | 认证类型 |“”|
| attend_events | 参加的事件id列表 |[]|
| wish_events | 感兴趣的事件id列表 |[23163156]|
